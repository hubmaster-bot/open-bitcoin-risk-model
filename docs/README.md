# OBRM Documentation

This directory contains the project documentation for the Open Bitcoin Risk Model.

## Documentation layers

### User Guide

For users who want to understand and operate OBRM without reading the code.

### Research Handbook

For readers who want to understand the model philosophy, calculations, assumptions, and limitations.

### Developer Guide

For contributors who want to understand the architecture, data flow, testing standards, and extension process.

### Architecture Decision Records

For understanding why major design decisions were made.
