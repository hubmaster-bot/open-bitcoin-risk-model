# Backtesting Lab

The Backtesting Lab compares fixed weekly DCA with OBRM dynamic allocation using the same external weekly cash contribution.

The dynamic strategy may buy more when cash has accumulated, retain cash, pause purchases, or gradually DCA out. Outputs include ending wealth, BTC holdings, cash, return on contributions, maximum drawdown, portfolio allocation, and historical action counts.

A backtest is not proof. Version 0.6.0 does not model transaction costs, taxes, slippage, custody risk, or imperfect execution.
