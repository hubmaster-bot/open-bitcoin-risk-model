# ADR-014: Use Model Confidence terminology

## Status

Accepted

## Decision

OBRM will use **Model Confidence** rather than the generic term Confidence.

## Meaning

Model Confidence answers:

> How much trust should be placed in the model's current assessment?

It does not mean:

- probability that Bitcoin will rise,
- probability that a trade will succeed, or
- certainty about future returns.

## Current factors

- Coverage
- Agreement
- Data Quality
- Model Maturity

Future releases may add Validation Strength and Regime Similarity.
