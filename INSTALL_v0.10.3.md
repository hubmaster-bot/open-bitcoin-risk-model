# OBRM v0.10.3 — Realized Capital Flow Research

## Upgrade

1. Replace the existing repository files with this release while preserving your local `data/` directory.
2. Create or activate the Python environment.
3. Install dependencies with `pip install -r requirements.txt`.
4. Run `pytest`.
5. Start with `streamlit run app.py`.
